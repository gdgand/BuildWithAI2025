# 참가자 안내사항

## Hands on Lab 준비

이번에 진행하는 Hands on Lab의 시스템/소프트웨어 적정/필수 요구 사양은 다음과 같습니다.

(필수 요구사항은 **(필수)** 태그를 함께 기재합니다)

- OS: 64bit MS Windows 10/macOS 12
- RAM: 16GB
- CPU: i5 8세대 CPU/macOS의 경우 M1 Apple Silicon 이상
- 디스크 공간: 최소 30기가 여유
- **(필수)** 안드로이드 스튜디오 버전: Android Studio Narwhal Canary 7
  - Hands on Lab 진행 시 [Gemini assistant에 사진 첨부 기능](https://developer.android.com/studio/preview/features#gemini-image-attachments)을 활용해야하기 때문에 해당 버전을 활용하셔야 합니다,

## Android Studio Narwhal 및 Jetbrains Toolbox 설치 안내

기존 안드로이드 안정버전을 사용하시던 분들에겐 2개 이상의 안드로이드 스튜디오를 운용하는 것이 불편함으로 느껴질 수도 있습니다. 이번 Android Studio 설치 안내에서는 2개 이상의 Android Studio를 더욱 편하기 쉽게 운용할 수 있는 방법인 Jetbrains Toolbox도 같이 소개를 드리고자 합니다.
